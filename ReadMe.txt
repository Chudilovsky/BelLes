Описание запуска тестового задания

Исользовались: -Oracle 11.2.0.2.0; Delphi 10.4

1) на компьютере с установленным клиентом Oracle в tnsnames.ora внести (если еще нет) алиас ORCL
	Примечание. Можно не создавать, а использовать другой, но, тогда при подключении п.2), п.3) и п.4.1) нужно будет указывать ваш существующий "Net Service Name"
2) В oracle создать тестового пользователя "testBel" с правами - с помощью скрипта "Oracle0.sql"
3) В oracle для тестового пользователя "testBel" создать необходимые объекты - с помощью скрипта "Oracle1.sql"
4) Запустить программу prBel.exe (можно взять готовую из "prBel_exe.rar"), или скомпилировать исходники из "prBel_Source.rar"
4.1) подключиться к СУБД Oracle - кнопка Oracle.
4.2) проверить функционал программы - кнопки "Фильтр, Поиск, FastRep, Export(Doc)", где 
 окошко слева - список оргранизаций
 окошко справа - список ключей организаций
 "Фильтр" - фильтрация списка ключей по наименованию(можно части слова) организации
 "Поиск" - поиск (позиционирование) в списке ключей по ключу
 "FastRep" - предварительный просмотр и печать списка ключей
 "Export(Doc)" - выгрузка списка ключей в файл MS Word с расширением ".docx"
